Informacje o projekcie

Przedstawiony projekt jest odwzorowaniem portalu YouTube.com Projekt został zrealizowany tak, aby wyglądem przypominał w jak największym stopniu znany portal z filmami

Instrukcja:

Po zamknięciu tego okna można włączyć API obsługujące wyszukiwarkę filmików, która działa według API udostępnionego przez YouTube. Filmy można wyszukiwać po nazwie lub wybierając jedną z kategorii z menu poniżej wyszukiwarki. Po kliknięciu na jeden z załadowanych po wyszukaniu filmików zostanie otwarte nowe okno z filmikiem, któremu odpowiada miniaturka.
